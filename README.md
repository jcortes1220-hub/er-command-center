# ER Manager Command Center

ER Manager Command Center is a lightweight workflow app built for daily manager use in the emergency department.

It helps with:

- daily, weekly, and monthly task tracking
- reminders and calendar exports
- editable contacts
- sticky notes
- carryover tasks
- shift-based task management for Morning, Mid, and Night

## Live App

Use the hosted version here:

[https://jcortes1220-hub.github.io/er-command-center/](https://jcortes1220-hub.github.io/er-command-center/)

## Works On

- Windows
- iPad
- Android

## How To Use

### Windows

Open the live link in Chrome or Edge.

### iPad

1. Open the live link in Safari.
2. Tap `Share`.
3. Tap `Add to Home Screen`.

### Android

1. Open the live link in Chrome.
2. Tap the browser menu.
3. Tap `Add to Home screen`.

## Main Features

- Today Snapshot with separate daily and weekly task selection
- Shift Board with moveable tasks
- Monthly progress tracking
- Carryover unfinished tasks to the next day
- Editable contacts
- Sticky notes with multiple note tabs
- Reminder export to calendar
- End-of-shift report preview with print-to-PDF option

## Important Note

Each device keeps its own saved data in the browser on that device.

That means:

- managers can all use the app
- they do not need GitHub login to open it
- their tasks and notes do not sync with each other

## Project Files

- `index.html` - main app
- `favicon.png` - browser icon
- `apple-touch-icon.png` - home screen icon
- `manifest.webmanifest` - mobile/home-screen app settings

## Updating The App

To make changes later:

1. Edit `index.html` locally.
2. Upload the updated file to the same GitHub repository.
3. Commit the change.
4. Wait a minute or two for GitHub Pages to refresh.

The live link stays the same.

## Future Upgrade Idea

If you want all managers to share the same live data, the app would need a backend and shared database.
