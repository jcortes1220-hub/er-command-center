# ER Command Center Share Package

This package lets other managers use the app on their own device.

## Files included

- `index.html`
- `favicon.png`
- `manifest.webmanifest`

## Desktop use

1. Put all 3 files in the same folder.
2. Open `index.html` in Chrome, Edge, or Safari.

## iPad use

Best option:

1. Upload the 3 files to a simple web host or shared website folder.
2. Open the app link in Safari on iPad.
3. Tap Share.
4. Tap `Add to Home Screen`.

This makes it behave more like an app on iPad.

## Important note

Each manager keeps their own tasks, notes, reminders, and contacts on their own device/browser.

This package does not sync data between managers.

## If you want shared live data

The app would need to be converted into an online app with a shared database.
